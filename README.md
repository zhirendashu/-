## 如何使用

### 方法一：本地使用
直接双击 `index.html` 用浏览器打开即可。

### 方法二：部署到 Cloudflare Pages（推荐）
1. 登录 https://dash.cloudflare.com
2. Workers & Pages → Create application → Pages → Upload Assets
3. 把本文件夹拖进去，点击 Deploy
4. 绑定你的自定义域名
